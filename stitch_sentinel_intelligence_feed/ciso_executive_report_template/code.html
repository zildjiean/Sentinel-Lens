Executive Summary

Current intelligence indicates a targeted escalation by APT29-DELTA actors against core cloud infrastructure. We have identified credential harvesting attempts targeting Level-1 Administrative accounts via sophisticated spear-phishing campaigns.

Immediate rotation of all privilege-access keys is required. Preliminary impact analysis suggests a potential exfiltration of non-encrypted legacy databases if no action is taken within the next 48 hours.
